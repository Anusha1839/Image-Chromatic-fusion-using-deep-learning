# Image-Chromatic-fusion-using-deep-learning

Project Overview
- Image Chromatic Fusion is a deep learning–based project that automates the colorization of grayscale images
- The project focuses on generating realistic and context-aware colors using Convolutional Neural Networks (CNN)
- Implemented using the PyTorch framework with a Gradio-based web interface
- Designed for historical photo restoration, media enhancement, and creative applications

# Objectives of the Project
+ Automatically convert grayscale images into color images
+ Learn complex color patterns and spatial features
+ Preserve texture, structure, and depth of original images
+ Provide a real-time, user-friendly web interface
+ Evaluate performance using standard image quality metrics

# Problem Definition
- Manual colorization is time-consuming and requires expertise
- Traditional methods fail on complex scenes and textures
- Lack of realism in conventional approaches
- Deep learning is used to overcome these limitations

# Proposed System
+ CNN-based feature extraction and color fusion
+ Predicts chromatic information from grayscale inputs
+ Produces high-quality, realistic colorized images
+ Real-time inference via web application

# Methodology
- Data collection from public datasets
- Image preprocessing including resizing and normalization
- CNN backbone for feature extraction
- Image Decoder and Color Decoder Block
- Fusion Module for final colorization
- Training using supervised learning
- Deployment using Flask and Gradio

# Dataset Used
- ImageNet and COCO datasets
+ Paired grayscale and color images
+ Images resized to fixed dimensions
+ Diverse textures and lighting conditions

# Model Evaluation Metrics
+ Peak Signal-to-Noise Ratio (PSNR)
+ Structural Similarity Index (SSIM)
+ Mean Squared Error (MSE)

# Results
- High PSNR and SSIM values achieved
- Realistic and vibrant colorized outputs
- Better performance than traditional models
- Preserved image structure and texture

# Web Application Features

- One-click colorization
- Real-time output display
- Interactive comparison view
